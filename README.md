# Portfolio
## Progetto - OOPLearning:

Progetto iniziato per poter consolidare le mie conoscenze sulla programmazione ad oggetti e per poter avere un'ambiente di test per leggere, trasformare e caricare file csv, xml, xlsx e json su db.

## Caratteristiche Principali

* **Configurazione Flessibile:** Gestione delle connessioni a database e altre impostazioni tramite file `config.ini`.
* **Caricamento Multi-Formato:** Supporto per l'importazione di dataset da:
    * `XML` (eXtensible Markup Language)
    * `CSV` (Comma Separated Values)
    * `XLSX` (Microsoft Excel Open XML Format Spreadsheet)
    * `JSON` (JavaScript Object Notation)
* **DataFrames Robusti:** Caricamento dei dati direttamente in strutture DataFrame per una manipolazione e analisi efficiente.
* **Processi ETL Integrati:** Funzionalità di Extract, Transform, Load per preparare e pulire i tuoi dataset.
* **Analisi Dati:** Strumenti integrati per l'analisi preliminare dei dati dopo il processo ETL.
* **Programmazione Orientata agli Oggetti (OOP):** Architettura modulare e scalabile per una facile manutenibilità ed estensione.
* **Logging Dettagliato:** Sistema di logging con timestamp per tracciare ogni operazione, facilitando il debug e il monitoraggio.

## Guida Rapida

### Installazione

1.  **Clona il repository:**
    ```bash
    git clone https://github.com/claudiaboccarossa/Portfolio.git
    cd tuo-repo
    ```
3.  **Installa le librerie:**
    ```bash
    pip install -r requirements.txt
    ```

### Configurazione

Il software si basa su un file di configurazione denominato `config.ini`. Seguire le istruzioni del .docx `istruzioni_file_ini`.
